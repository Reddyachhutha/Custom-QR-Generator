About Me

Hi, I'm REDDY ACHHUTHA! Welcome to my project. This is a highly customizable QR Code Generator designed to bring creativity into static links. You can generate custom patterns, add central logos, and blend images directly into the QR code data points.

GitHub LinkedIn

Features

  • 🎨 Custom Module Shapes (Circles, Rounded, Squares)
  • 🖼️ Logo Integration (Shape masking, dynamic sizing, and positioning)
  • 🌄 Background Image Blending with Smart Contrast Adjustment
  • 🎯 Guaranteed Scannability (Auto-solidifies finder patterns)
  • 🖱️ Interactive UI (Draggable QR code block, glassmorphism design)
  • ⚡ FastAPI Backend for lightning-fast generation
